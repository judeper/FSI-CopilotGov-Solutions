### F1 — Sensitivity label coverage overstates Copilot restriction

- **Severity:** Major
- **Location:** lines 61-73
- **Current:** "Sites without labels may expose regulated content via Microsoft 365 Copilot."
- **Issue:** The evidence description implies site label presence or absence directly controls Copilot exposure. Microsoft documentation says Copilot honors labels when configured for protection; for encrypted labels, users must have VIEW and EXTRACT usage rights for AI apps to return data.
