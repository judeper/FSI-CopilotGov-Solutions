### F1 — SharePoint Advanced Management licensing path is incomplete

- **Severity:** Major
- **Location:** lines 42, 48
- **Current:** "Confirm SharePoint Advanced Management licensing and DSPM for AI prerequisites are available for the target tenant."
- **Issue:** The README does not mention that current SharePoint Advanced Management prerequisites can be satisfied by either Microsoft 365 Copilot license entitlement or standalone Microsoft SharePoint Advanced Management, after base licensing requirements are met.

### F2 — DSPM for AI should be labeled classic or replaced with current DSPM

- **Severity:** Major
- **Location:** lines 42, 48
- **Current:** "Confirm SharePoint Advanced Management licensing and DSPM for AI prerequisites are available for the target tenant."
- **Issue:** Microsoft Learn now describes DSPM for AI as the classic experience and directs customers to current Microsoft Purview Data Security Posture Management for the new version with broader AI app and agent support.
- **Suggested change:** Use Microsoft Purview Data Security Posture Management (DSPM) for current guidance, or explicitly say DSPM for AI (classic) when the classic workflow is intended.
- **Source:** https://learn.microsoft.com/purview/dspm-for-ai (last updated 2025-12-15)
- **Quote:** "This article is for the classic version of Data Security Posture Management for AI that's now replaced."

### F3 — Restricted SharePoint Search caveats are missing

- **Severity:** Major
- **Location:** lines 20, 82, 113
- **Current:** "Restricted SharePoint Search enablement planning for tighter Copilot grounding boundaries."
- **Issue:** The README presents Restricted SharePoint Search as a grounding-boundary control without the current Microsoft caveats that it is temporary, limited, not a security boundary, and does not change SharePoint permissions.
- **Suggested change:** Add a caveat that Restricted SharePoint Search is a temporary measure with up to 100 allowed sites, is not a security boundary, does not change permissions, and does not guarantee only allowed-list sites appear in search or Copilot responses.
- **Source:** https://learn.microsoft.com/sharepoint/restricted-sharepoint-search (last updated 2026-02-24)
- **Quote:** "Restricted SharePoint Search isn't a security boundary and doesn't change any permissions on SharePoint sites."
