## Findings

### F1 — Graph featureRolloutPolicies are Entra staged authentication policies, not Copilot feature controls

- **Severity:** Critical
- **Location:** lines 21, 45-47, 83, 108, 171
- **Current:** "`Microsoft Graph beta rollout policies` and `/policies/featureRolloutPolicies` are listed as Copilot feature inventory and rollout sources."
- **Issue:** Microsoft Learn documents featureRolloutPolicy as a Microsoft Entra ID staged rollout resource for authentication-related features. Using it as a Copilot feature inventory or ring-management source would misdirect implementers to the wrong API.
- **Suggested change:** "Remove `/policies/featureRolloutPolicies` from Copilot FMC source-system guidance, or clearly scope it to Entra staged authentication only. For Copilot feature governance, reference documented Microsoft 365 admin center, Teams meeting policy, Cloud Policy, and Power Platform admin center controls."
- **Source:** https://learn.microsoft.com/en-us/graph/api/resources/featurerolloutpolicy?view=graph-rest-beta (last updated 2024-07-06)
- **Quote:** "pilot features of Microsoft Entra ID with a specific group before enabling features for entire organization"

### F2 — Policy.ReadWrite.FeatureRollout is not the current featureRolloutPolicies permission

- **Severity:** Major
- **Location:** lines 132-135
- **Current:** "`Policy.ReadWrite.FeatureRollout` is listed as required delegated access / Graph scope."
- **Issue:** The current Microsoft Graph permissions table for listing featureRolloutPolicies uses HybridAuthentication permissions, not `Policy.ReadWrite.FeatureRollout`. The listed permission is stale and would confuse implementation planning even before accounting for the Copilot API mismatch.
- **Suggested change:** "Do not list `Policy.ReadWrite.FeatureRollout` for Copilot governance. If documenting Entra staged rollout separately, use the current Microsoft Graph HybridAuthentication permissions from Learn."
- **Source:** https://learn.microsoft.com/en-us/graph/api/featurerolloutpolicies-list?view=graph-rest-beta (last updated 2024-03-06)
- **Quote:** "Delegated (work or school account) Policy.Read.HybridAuthentication"
