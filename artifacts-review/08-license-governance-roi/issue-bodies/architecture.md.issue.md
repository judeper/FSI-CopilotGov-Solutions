## Findings

### F1 — Update Copilot usage report endpoints

- **Severity:** Critical
- **Location:** lines 14 and 64-65
- **Current:** "`/beta/reports/...`; `GET https://graph.microsoft.com/beta/reports/getMicrosoft365CopilotUsageUserCounts(period='D30')`"
- **Issue:** The documented `getMicrosoft365CopilotUsageUserCounts` endpoint was not found, and current Microsoft 365 Copilot usage report APIs use `/v1.0/copilot/reports/...` or `/beta/copilot/reports/...` names such as UserCountSummary and UserCountTrend.
- **Suggested change:** "Replace the endpoints with `GET https://graph.microsoft.com/v1.0/copilot/reports/getMicrosoft365CopilotUsageUserDetail(period='D30')`, `getMicrosoft365CopilotUserCountSummary`, and/or `getMicrosoft365CopilotUserCountTrend`."
- **Source:** https://learn.microsoft.com/microsoft-365/copilot/extensibility/api/admin-settings/reports/resources/copilotreportroot (last updated 2026-01-08)
- **Quote:** "getMicrosoft365CopilotUserCountSummary getMicrosoft365CopilotUserCountTrend getMicrosoft365CopilotUsageUserDetail"

### F2 — Do not model per-user Copilot action counts as Graph report data

- **Severity:** Major
- **Location:** line 78
- **Current:** "`CopilotUsageDetail` example fields include `CopilotActions30D`."
- **Issue:** Current Microsoft Learn notes that per-user Copilot prompt counts across tenants are not supported through the Graph report API; the API returns usage/last-activity detail, not a per-user action-count metric.
- **Suggested change:** "Model last activity dates and active/enabled status from Graph reports; use admin-center exports, Purview audit, or clearly labeled local metrics for per-user prompt/action counts."
- **Source:** https://learn.microsoft.com/microsoft-365/copilot/extensibility/api/admin-settings/reports/copilotreportroot-getmicrosoft365copilotusageuserdetail (last updated 2026-02-19)
- **Quote:** "Tracking per-user Copilot prompt counts across tenants isn't supported due to privacy and security constraints."

- **Source:** https://learn.microsoft.com/microsoft-365/copilot/pay-as-you-go/setup (last updated 2025-11-18)
