# Common Prerequisites

- Microsoft 365 licensing appropriate for Microsoft 365 Copilot and the target compliance features
- Administrative access to Microsoft 365 Admin Center, Microsoft Purview, Microsoft Entra, and Power Platform where applicable
- PowerShell 7 or later and Python 3.11 or later for local script execution
- A documented owner list for platform, identity, compliance, operations, and change management
- An approved secret or certificate storage approach for deployment identities, connector credentials, and notification endpoints
- A documented decision on governance tier: baseline, recommended, or regulated
- An evidence-retention location for exported solution packages
- A non-production validation plan and production change window strategy

Use [Identity and Secrets Prep](./identity-and-secrets-prep.md) to convert these prerequisites into a named operator preflight record.
